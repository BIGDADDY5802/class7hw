# using vim

*   read testfiles with vim 

*   exit vim = esc :q! (quit) :wq (write and quit)

*   cat in conjunction with head and tail commands:
    head is the beginning of the files lines / ex: cat testfile | head -n5

    cat in conjunction with tail is end of file / ex: cat testfile | tail -n5

we can also use $cat im conjuntion with "more" or "less" ex: cat testfile1 | more etc...
