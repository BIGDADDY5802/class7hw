# pipe

*   "|" = pipe

*       command1 | command2

*          cat testfile1.txt | testfile2

*           ls -l | sort wc -l

Pipes are a fundamental and highly effective tool in the Linux command-line environment, enabling users to combine simple commands into sophisticated and efficient workflows

> = add to file
>> = append file
< = redirect reading file | redirect adding contents

ls -al >> testfile3.txt = this command would print the output of cmd into testfile3.txt
